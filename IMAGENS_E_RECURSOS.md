## 👥 Avatares para Depoimentos

### Opções de Busca
```
Unsplash: "professional headshot"
Pexels: "portrait people business"
Pixabay: "corporate avatar"
```

### Características
- Foto lateral ou frontal
- Sorriso natural
- Fundo limpo
- Resolução: 400x400px ou maior
- Formato: JPEG ou PNG

### Alternativa: Usar Initials
Se não achar fotos, use avatares com iniciais:
```jsx
<div className="w-12 h-12 bg-gradient-neon rounded-full 
               flex items-center justify-center text-xl font-bold">
  AS {/* Ana Silva */}
</div>
```

## 📊 Dimensões Recomendadas

### Imagens
- **Hero**: 1920x1080 (16:9)
- **Cards**: 600x400 (3:2)
- **Galeria**: 800x600 (4:3)
- **Avatares**: 200x200 (1:1)

### Otimização
```
JPG: Para fotos (60-80% qualidade)
PNG: Para logos/ícones
WebP: Para web (melhor compressão)
```

### Ferramentas de Otimização
- **TinyPNG** (tinypng.com) - Compressão
- **Squoosh** (squoosh.app) - Google tool
- **ImageOptim** - Para Mac
