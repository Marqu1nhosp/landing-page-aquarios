## 🚀 Como Usar

### 1️⃣ Instalação (1 minuto)
```bash
cd f:\Projetos\landing-page-aquarios
npm install
```

### 2️⃣ Desenvolvimento (1 minuto)
```bash
npm run dev
```
Abra: **http://localhost:5173**

### 3️⃣ Customizar (5 minutos)
- Nome: `src/components/Navbar.jsx` (linha 40)
- Cores: `tailwind.config.js` (linhas 15-20)
- Textos: `src/components/*.jsx` (arrays de dados)
- Imagens: `src/components/Hero.jsx` e `Gallery.jsx`

### 4️⃣ Deploy (5 minutos)
```bash
npm run build    # Gera pasta dist
vercel          # ou netlify deploy
```
